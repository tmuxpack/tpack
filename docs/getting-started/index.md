# Getting Started

Get up and running with tpack in under 5 minutes.

---

**1. [Install](installation.md)** — Pick your platform and install tpack.

**2. [Configure](configuration.md)** — Add plugins to your `tmux.conf` and load tpack.

**3. [Use](../usage/index.md)** — Manage plugins with key bindings, the TUI, or the CLI.

---

Already using TPM? See [Migrating from TPM](migrating-from-tpm.md) — tpack is a drop-in replacement, so you likely don't need to change anything.
