# Usage

tpack offers three ways to manage plugins: key bindings, the interactive TUI, and the CLI.

---

**[Managing Plugins](managing-plugins.md)** — Install, update, and uninstall plugins using tmux key bindings.

**[Interactive TUI](interactive-tui.md)** — Browse plugins, run operations, and inspect commit history in a terminal UI.

**[CLI Reference](cli-reference.md)** — Use the `tpack` binary directly from the command line.

**[Automatic Updates](automatic-updates.md)** — Configure background update checking and self-update for the tpack binary.
